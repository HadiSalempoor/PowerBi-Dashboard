# Power BI Dashboard | Business Performance Analysis

This project analyzes business performance using an interactive Power BI dashboard. It focuses on KPIs, customer trends, operational performance, and data-driven decision-making.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Key Highlights
- Built an interactive dashboard to monitor business performance.
- Analyzed KPIs, customer trends, and operational metrics.
- Designed visuals to support executive-level decision-making.
- Used real-world business data to identify actionable insights.
